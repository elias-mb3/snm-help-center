Adicionar um Serviço

Agora, para adicionar um serviço no sistema é bem simples, basta clicar no botão “ + Criar Serviço”.

Nessa modal que se abriu ao clicar no botão, podemos ver as opções para adicionar foto ao serviço, preço do serviço, comissão padrão do serviço, duração, status e descrição.

Para exemplificar vou criar um serviço de manicure:

  • Título: Manicure
  • Preço: 50%
  • Duração: 00:30
  • Status: Ativos
