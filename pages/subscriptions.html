Serviços

Clientes

Na Página de clientes, você pode adicionar novos clientes e visualizar todos os seus clientes.

Adicionar um Serviço

Para adicionar um novo cliente basta clicar em “+ Criar Cliente”, irar abrir uma modal para adicionar os dados dos clientes. Os dados obrigatórios para criar um usuário são Nome e Telefone, os demais dados não são obrigatórios mais são muito importantes para os relatórios.

Nesse Exemplo vou criar um novo cliente:

  • Nome: Monica
  • Telefone: (15) 9 3778-6607

Endereço

Para adicionar as opções de endereço podemos adicionar apenas o CEP ou habilitar a opção de endereço manual para complementar os dados como Número.

Outras Informações

Nessas opções temos outros dados demográficos importante para os relatórios e informações de estatisticas “Como nos conheceu?” e a possibilidade de adicionar observações sobre esse cliente.
